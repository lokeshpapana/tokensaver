Free API Access

Use the REST API with rate limits. No API key = 20 req/hr. Free key = 100 req/hr.

# Get a free API key curl -X POST "http://localhost:8000/api/keys/generate?tier=free&description=my-key" # Convert a file (anonymous, 20 req/hr) curl -X POST http://localhost:8000/api/convert -F "file=@myfile.py" # Convert with API key (100 req/hr) curl -X POST http://localhost:8000/api/convert \ -H "X-API-Key: YOUR_KEY_HERE" -F "file=@myfile.py" # Convert pasted text directly curl -X POST http://localhost:8000/api/convert-text \ -H "Content-Type: application/json" \ -d '{"text": "your code here", "filename": "code.py"}' # Batch convert (returns ZIP) curl -X POST "http://localhost:8000/api/convert-batch?mode=dense" \ -F "files=@file1.py" -F "files=@file2.json" -o batch.zip # View history curl http://localhost:8000/api/history # Check your rate limit curl http://localhost:8000/api/keys/check -H "X-API-Key: YOUR_KEY_HERE"
